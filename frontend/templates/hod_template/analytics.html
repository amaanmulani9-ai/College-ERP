    Prometheus Exporter Endpoint

    Endpoint: /metrics

    This provides users count metrics, fee totals, placement ratios, attendance metrics, and library details.

    Grafana Dashboard Interface

    A comprehensive dashboard configuration template has been pre-configured and saved inside this project.

    File Path: monitoring/grafana_dashboard.json

    Import this json file directly into your Grafana instance to monitor real-time dashboards for Students, Staff, Finance, and Placement telemetry.

    Quick Start: Run Monitoring Stack (Prometheus + Loki + Promtail)

    Follow these steps to run the monitoring stack on your local machine:

    1. Install Docker or download Prometheus, Grafana Loki, and Promtail binaries.
    2. Start Prometheus using the configuration file in this project: prometheus --config.file=monitoring/prometheus.yml.
    3. Start Loki: loki --config.file=monitoring/loki-config.yml.
    4. Start Promtail to ship logs: promtail --config.file=monitoring/promtail-config.yml.
    5. Open Grafana (port 3000), add Prometheus and Loki as data sources, and import the monitoring/grafana_dashboard.json.
